Dashboard menampilkan ringkasan informasi sistem, seperti jumlah buku, total anggota, serta total peminjam. Dari halaman ini, admin dapat mengakses seluruh menu pengelolaan.
<img width="1600" height="749" alt="image" src="https://github.com/user-attachments/assets/3d4e62bd-0983-48ca-9106-b71cc9dc884f" />
Halaman Tambah buku untuk menampilkan daftar buku yang tersedia di perpustakaan. Admin dapat menambah, mengubah, menghapus, dan mencari data buku berdasarkan judul atau kategori.
<img width="1600" height="743" alt="image" src="https://github.com/user-attachments/assets/70f96474-2467-446f-984d-27917640597b" />
Halaman Peminjaman buku digunakan untuk melakukan transaksi peminjaman buku oleh anggota perpustakaan. Petugas dapat memproses peminjaman dengan memilih anggota yang valid dan buku yang tersedia, serta mencatat tanggal peminjaman untuk keperluan tracking dan pengembalian.
<img width="1600" height="737" alt="image" src="https://github.com/user-attachments/assets/e5295f70-f81d-4914-96e3-dd1464a517ea" />
Halaman Pengembalian buku digunakan untuk memproses pengembalian buku yang telah dipinjam. Petugas dapat memasukkan ID peminjaman untuk mencatat pengembalian, menghitung denda keterlambatan jika ada, dan mengupdate status ketersediaan buku di sistem.
![Uploading image.png…]()
Halaman Laporan peminjam digunakan untuk menampilkan seluruh data transaksi peminjaman buku dalam sistem. Petugas dapat melihat riwayat lengkap peminjaman, status pengembalian, dan informasi denda untuk keperluan monitoring dan pelaporan aktivitas perpustakaan.
![Uploading image.png…]()
Halaman Tambah Anggota digunakan untuk mengelola data anggota perpustakaan. Admin dapat menambahkan anggota baru dengan mengisi informasi lengkap, serta melihat, mengedit, atau menghapus data anggota yang sudah terdaftar dalam sistem.
